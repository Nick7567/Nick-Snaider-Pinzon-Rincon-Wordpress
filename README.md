# Nick-Snaider-Pinzon-Rincon-Wordpress
# Descripción del proyecto
Para este proyecto desarrollé una página web dedicada a las noticias de artes marciales, enfocada especialmente en karate, judo, Brazilian Jiu-Jitsu (BJJ), Muay Thai y MMA. El sitio, fue creado utilizando WordPress con el objetivo de ofrecer un portal informativo donde los usuarios puedan consultar novedades, eventos, competiciones y artículos relacionados con estas disciplinas.

# Instalación y configuración
Comencé instalando el entorno LAMP y posteriormente configuré WordPress. Despues seleccione un buen estilo visual para tener una buena presentación visual sobre las noticias de las artes marciales.
Luego personalicé el diseño ajustando elementos como el menú principal, la paleta de colores, la estructura de la página de inicio y las secciones internas, buscando una experiencia intuitiva para los aficionados a las artes marciales.

# Organización del contenido
Para estructurar correctamente la información, creé varias categorías específicas para cada disciplina. Algunas de ellas son:

# Karate

# Judo

# Brazilian Jiu-Jitsu (BJJ)

# Muay Thai

# Mixed Martial Arts (MMA)

También publiqué artículos de prueba con títulos, imágenes destacadas, etiquetas y contenido simulado

# Configuración del sitio
Durante el desarrollo configuré varios elementos esenciales del sitio web:

Creando cada post para diferente arte marcial

Un poco de categorias para un arte marcial

Agregar un mapa en donde muestra en donde fue el debut en Mixed Martial Arts (MMA)

Un apartado de idiomas en el que puedas seleccionar una cierta variedad de idiomas para que puedas ver la pagina en tu respectivo idioma
